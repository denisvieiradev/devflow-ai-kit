# devflow-ai-kit

CLI pipeline for structured software development — from PRD to merge.

## Install

```bash
npm install -g devflow-ai-kit
```

## Quick Start

```bash
cd my-project
devflow init
devflow prd "add OAuth authentication"
devflow techspec 001
devflow tasks 001
devflow run-tasks 001
devflow review 001
devflow pr
devflow done 001
```
